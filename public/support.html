Aide et support

Dabt aide a organiser le hifz quotidien, a suivre les erreurs page par page et a recevoir des rappels de revision.

  • Si tu ne recois pas les rappels, verifie que les notifications Android sont autorisees.
  • Sur certains appareils, Android peut demander une autorisation supplementaire pour les alarmes exactes.
  • La progression est locale: une desinstallation ou une reinitialisation peut supprimer les donnees enregistrees.
  • Pour les questions produit, les retours ou les demandes liees a la confidentialite, utilise le formulaire GitHub ci-dessus ou l'adresse de contact du developpeur affichee sur Google Play.

Help and support

Dabt helps organize daily hifz, track mistakes page by page, and schedule review reminders.

  • If reminders do not fire, check that Android notifications are allowed.
  • On some devices, Android may ask for an additional exact alarm permission.
  • Progress is stored locally, so uninstalling or resetting the app can remove saved data.
  • For product questions, feedback, or privacy-related requests, use the GitHub form above or the developer contact details shown on Google Play.

الدعم والمساعدة

يساعد Dabt على تنظيم الحفظ اليومي، وتتبع الأخطاء صفحة بصفحة، وجدولة تذكيرات المراجعة.

  • إذا لم تصل التذكيرات، فتأكد من السماح بإشعارات Android.
  • قد تطلب بعض الأجهزة إذنا إضافيا للتنبيهات الدقيقة.
  • التقدم محفوظ محليا، لذلك قد يؤدي حذف التطبيق أو إعادة ضبطه إلى حذف البيانات المحفوظة.
  • للاستفسارات أو الملاحظات أو الطلبات المتعلقة بالخصوصية، استخدم نموذج GitHub أعلاه أو بيانات المطور المعروضة في Google Play.
